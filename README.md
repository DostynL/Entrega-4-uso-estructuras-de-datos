## Descripción
Sistema de análisis académico desarrollado en Julia que combina múltiples 
estructuras de datos (Dict, Array, NamedTuple, Set, Matrix) para gestionar 
estudiantes, cursos y calificaciones.

## Estructuras utilizadas
- `NamedTuple` – Representar datos de cada estudiante
- `Dict` – Organizar estudiantes y cursos
- `Array` – Almacenar y operar calificaciones
- `Set` – Gestionar cursos únicos
- `Matrix` – Tabla general de notas

## Requisitos
- Julia 1.12+
- VS Code con extensión Julia

## Cómo correr
```bash
julia sistema_academico.jl
```
